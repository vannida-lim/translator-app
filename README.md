**Tech Stack**

-   React
-   Google Translate API Key 

**Installing**
--------------
-   Clone this repo to your local machine `git clone git <this-repo-url>`

-   `cd`  to `translator-app`

-   run  `npm i && npm start` to run dependencies and start server locally
